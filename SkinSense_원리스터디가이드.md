## 1. 전류, 저항, 임피던스 — 기초부터

### 1-1. 직류(DC) 저항: 가장 쉬운 개념

**비유: 물이 파이프를 흐르는 것처럼**

```
전압(V) = 수압 (물을 밀어내는 힘)
전류(I) = 유량 (실제 흐르는 물의 양)
저항(R) = 파이프 좁기 (흐름을 방해하는 것)

공식: V = I × R    (옴의 법칙)
      전압 = 전류 × 저항
```

> 예시: 1.5V 건전지에 저항 100Ω을 연결하면 전류 = 1.5V ÷ 100Ω = 0.015A (15mA)

---

### 1-2. 교류(AC)와 임피던스: 파도치는 전류

**직류(DC)**: 한 방향으로만 흐름 (건전지, USB 충전기 내부)
**교류(AC)**: 방향이 주기적으로 바뀜 (가정용 220V, 코일 구동에 사용)

```
교류의 주파수(f): 초당 방향 전환 횟수
  - 가정용 전기: 60Hz (1초에 60번 방향 전환)
  - 우리 센서:   1MHz ~ 100MHz (1초에 백만~1억 번!)
```

**임피던스(Z)**: 교류 회로에서의 '총체적 방해'

```
Z = R + jX      (복소수로 표현)
    │    │
    │    └ 리액턴스 (X): 코일/커패시터로 인한 방해
    └ 저항 (R): 직류에서와 같은 방해

|Z| = √(R² + X²)    (임피던스 크기)
φ = arctan(X/R)     (위상각)
```

**왜 R과 |Z|가 이렇게 다른가?**

```
우리 코일의 예:
  R  = 0.742 Ω    (구리선 자체의 저항, 아주 작음)
  X  = 474 Ω      (코일 인덕턴스에 의한 리액턴스, 매우 큼!)
  |Z|= √(0.742² + 474²) ≈ 474 Ω

→ 교류에서는 코일의 '자기적 저항'(리액턴스)이 지배적!
→ 피부 수분에 의해 R이 변하면 → |Z|와 위상각(φ)이 변화
→ 이 변화를 측정하는 것이 SkinSense의 핵심!
```

---

### 1-3. 인덕턴스 (L): 코일의 핵심 특성

```
코일에 전류가 흐르면 → 자기장 생성
전류가 변하면 → 자기장도 변함
변하는 자기장 → 전압을 역방향으로 유도 (렌츠 법칙)

이 '자기적 관성'을 인덕턴스(L)라 부름
단위: 헨리(H), 마이크로헨리(μH)

인덕턴스 영향:
  - 주파수 높을수록 리액턴스 커짐: X_L = 2π × f × L
  - 코일 주변에 전도성 물체(피부) 가까워지면 → L이 변함 → Z가 변함
  → 이것이 센서의 기본 동작 원리!
```

## 2. 맴돌이전류(Eddy Current) 원리

### 2-1. 패러데이 법칙: 변화하는 자기장이 전류를 만든다

```
[패러데이 전자기 유도 법칙]

변화하는 자기장
      ↓
도체 내부에 전압(EMF) 유도
      ↓
도체가 있으면 전류 흐름

시각화:
  코일 (교류 전류) → 변하는 자기장 →┐
                                    │ (도체: 피부, 금속)
  ←── 역방향 자기장이 코일로 되돌아옴 ←── 맴돌이전류 발생!
```

### 2-2. 맴돌이전류란 무엇인가?

```
이름의 유래: 도체 내부에 소용돌이처럼 순환하는 전류

[발생 과정]
1. 코일에 교류 전류 흘림 (우리: 100MHz 고주파)
2. 변화하는 자기장이 코일 앞의 피부에 도달
3. 패러데이 법칙에 의해 피부 내부에 작은 폐루프 전류 유도
4. 이 '맴돌이전류'가 역자기장을 만들어 코일 임피던스를 변화시킴

      [코일]         [피부 단면]
         │          ┌──────────────┐
     자기장 →→→→→→  │  맴돌이전류  │
         │          │  ↺  ↺  ↺   │
         ↑          │  ↺  ↺  ↺   │
    역자기장 ←←←←←  │             │
                    └──────────────┘
```

### 2-3. 피부 수분과 맴돌이전류의 관계

```
피부 수분 많음 → 전기 전도도(σ) 높음 → 맴돌이전류 강하게 흐름
                                         → 코일 임피던스 더 많이 변화

피부 건조      → 전기 전도도(σ) 낮음 → 맴돌이전류 약하게 흐름
                                         → 코일 임피던스 변화 작음

즉: 임피던스 변화량 = 피부 수분 지표!

수분 - 전도도 관계:
  건조 피부(각질층):  σ ≈ 0.0001~0.001 S/m
  수분 찬 피부:       σ ≈ 0.1~2.0 S/m
  생리식염수(0.9%):   σ ≈ 1.5 S/m
  물:                 σ ≈ 0.0005 S/m (순수한 물은 낮음, 이온이 전도체!)
```

## 3. 스킨 효과(Skin Effect)와 침투 깊이

### 3-1. 스킨 효과란?

```
고주파 교류에서 전류가 도체 표면에 집중되는 현상

비유: 고속도로 출퇴근 시간
  저주파(저속): 차들이 모든 차선 고르게 분산
  고주파(고속): 차들이 바깥 차선(표면)에만 집중!

원인: 고주파 자기장이 도체 내부에서 스스로 상쇄됨
      → 전류는 표면 근처에만 흐를 수 있게 됨
```

### 3-2. 침투 깊이 공식: SkinSense의 핵심 수식

$$\delta = \frac{1}{\sqrt{\pi f \mu \sigma}}$$

```
δ  = 침투 깊이 (m) — 전류가 표면의 1/e (≈37%)로 감소하는 깊이
f  = 주파수 (Hz)   — 높을수록 얕아짐
μ  = 투자율 (H/m)  — 피부: μ ≈ μ₀ = 4π×10⁻⁷ H/m (비자성체)
σ  = 전기 전도도 (S/m) — 수분이 많을수록 높아짐
```

**피부에서의 침투 깊이 계산 예시**

피부 전도도 σ = 0.2 S/m, μ = μ₀ 가정:

| 주파수 | 침투 깊이(δ) | 측정 대상 층 |
|--------|-------------|------------|
| 100 kHz | ~1,125 μm (1.1mm) | 표피 + 진피 상부 |
| 1 MHz   | ~356 μm | 표피 전층 |
| 10 MHz  | ~113 μm | 표피 하부 |
| 100 MHz | ~35 μm  | **각질층** (10~20μm) ← 우리 타겟! |
| 500 MHz | ~16 μm  | 각질층 최상부 |

```
[결론]
100MHz → δ ≈ 35μm → 각질층(두께 10~20μm)에만 맴돌이전류 유도!
1MHz   → δ ≈ 356μm → 진피층까지 침투

이 원리로 주파수를 바꾸면 측정 깊이를 선택할 수 있음!
```

### 3-3. 왜 100MHz인가? (SkinSense의 주파수 선택 근거)

```
각질층 두께: 약 10~20 μm

δ ≤ 20 μm 가 되려면:
  δ = 1/√(π × f × μ₀ × σ) ≤ 20×10⁻⁶

  f ≥ 1/(π × μ₀ × σ × (20×10⁻⁶)²)
  f ≥ 약 50 MHz

→ 최소 50MHz 이상 필요
→ 100MHz: 침투깊이 ~35μm → 각질층 포함, 적당한 신호 강도
→ 더 높은 500MHz+: 더 얕지만 회로 구현 어려움, 비용↑

특허 청구항에서 '50MHz 이상'으로 설정한 물리적 근거!
```

## 4. 피부의 전기적 특성

### 4-1. 피부 층별 구조와 전기적 특성

```
피부 단면 (실제 두께 비율과 다름, 이해용):

표면 (공기)
──────────────────────────────  ← 코일에서 1~5mm
각질층 (Stratum Corneum)
  두께: 10~20 μm
  수분: 10~20% (건조) ~ 30%+ (촉촉)
  전도도: 0.0001~0.001 S/m (매우 낮음)
  역할: 피부 장벽, 수분 증발 방지
──────────────────────────────
표피 (Epidermis)
  두께: 50~150 μm
  전도도: 0.01~0.1 S/m
──────────────────────────────
진피 (Dermis)
  두께: 1~4 mm
  혈관, 림프관, 수분 풍부
  전도도: 0.5~2.0 S/m (높음)
  역할: '속보습' 반영
──────────────────────────────
피하지방
```

### 4-2. 수분과 전기 전도도의 관계

```
피부 전도도 ≈ 이온 농도 × 이온 이동도

물 자체는 전도도 낮음 (순수물: 0.000001 S/m)
하지만 피부 속 물에는 Na⁺, K⁺, Cl⁻ 이온이 녹아 있음
→ 수분 ↑ = 이온 채널 ↑ = 전도도 ↑

피부 수분 - 전도도 관계 (근사):
  σ [S/m] ≈ 0.001 × exp(0.05 × M)   [M = 수분 함량 %, 단순화]

실제 측정값:
  Corneometer 20 AU (건조) → σ ≈ 0.0001~0.001 S/m
  Corneometer 60 AU (촉촉) → σ ≈ 0.01~0.1 S/m
  Corneometer 90 AU (매우 촉촉) → σ ≈ 0.1~0.5 S/m
```

### 4-3. 유전율(Dielectric Constant)과 공진 주파수

```
물의 비유전율: ε_r ≈ 80 (매우 높음!)
건조 각질층:   ε_r ≈ 2~5
수분 찬 각질층: ε_r ≈ 20~40

LC 공진 주파수 공식:
  f₀ = 1 / (2π√(LC))

  피부 수분 증가 → ε_r 증가 → 등가 커패시턴스 C 증가
                              → f₀ = 1/(2π√LC) 감소!
                              → 공진 주파수가 낮아짐

→ 공진 주파수 변화를 측정 = 피부 수분 측정!
   (주파수 시프트 측정 방식 — 초보자핸드북의 74HC14 회로가 이 원리)
```

## 5. 왜 복수 주파수인가? — 겉보습 vs 속보습

### 5-1. 화장품 업계의 핵심 니즈

```
겉보습 (표면 보습):
  - 각질층의 수분 함량
  - 즉각적인 보습감
  - Corneometer가 측정하는 것
  - 보습 크림 바른 직후 수치 높아짐

속보습 (심층 보습):
  - 진피층의 수분 함량
  - 장기적인 피부 건강
  - 기존 Corneometer로는 측정 불가능!
  - 히알루론산 등 '속보습' 성분의 효과 지표

기존 장비의 한계:
  Corneometer → 각질층만 측정 (접촉식, 겉보습만)
  SkinSense   → 복수 주파수로 겉보습 + 속보습 동시 측정!
```

### 5-2. 복수 주파수 차분법: 신호 분리의 핵심

```
측정값:
  Z_고주파 (100MHz) = f(각질층 수분 + 진피층 수분×α₁)
  Z_저주파 (1MHz)   = f(각질층 수분×β + 진피층 수분)

여기서 α₁ << 1 (고주파는 진피층에 거의 도달 못 함)
      β   << 1 (저주파의 각질층 기여는 상대적으로 작음)

각질층 순수 신호 추출:
  S_겉보습 = Z_고주파 - α × Z_저주파
             ↑ 고주파 측정에서 저주파의 심층 기여분 제거

진피층 순수 신호:
  S_속보습 = Z_저주파 - β × Z_고주파
             ↑ 저주파 측정에서 고주파의 표면 기여분 제거

[특허 청구항 1의 핵심 수식]:
  S_sc = Z₁ - (α × Z₂)    [α = 0.1~0.3]
```

### 5-3. 시각적 이해

```
고주파 (100MHz, δ≈35μm):
각질층 ▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓  → 강하게 감지
표피   ░░░░░░░░░░░░░░░     → 약하게 감지
진피   ·····················  → 거의 감지 안 됨

저주파 (1MHz, δ≈356μm):
각질층 ▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓  → 감지
표피   ▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓  → 강하게 감지
진피   ▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓  → 강하게 감지

차분 계산:
  고주파 - (α × 저주파) = 각질층 순수 신호만 추출!
```

## 6. 임피던스 평면 분석 (Impedance Plane)

### 6-1. 2차원으로 임피던스를 표현하는 이유

```
임피던스는 복소수: Z = R + jX

R (실수부): 저항 성분 — 에너지 소산
X (허수부): 리액턴스 성분 — 에너지 저장

2D 평면에 찍으면:
  X축: R (저항)
  Y축: X (리액턴스)

측정 포인트가 이 평면 위 어디에 있느냐가 피부 상태를 나타냄!
```

### 6-2. 임피던스 평면 해석

```
임피던스 평면 (NDT에서 차용):

  X (리액턴스)
  ↑
  │  ○ 공기 중 (기준점)
  │   \
  │    ● 건조 피부
  │     \
  │      ● 보통 피부
  │       \
  │        ● 촉촉한 피부
  │         \
  └──────────────────→ R (저항)

- 수분 증가: 저항(R) 감소 + 리액턴스(X) 감소 → 왼쪽 아래로 이동
- 이격 거리 증가: 신호 전체 약해짐 → 기준점 쪽으로 이동
- 온도 상승: R 감소 → 오른쪽으로 이동 (보정 필요!)

이 평면에서 궤적이 달라지므로 수분/거리/온도를 동시에 분리 가능!
```

## 7. AI 보정 알고리즘 원리

### 7-1. 왜 보정이 필요한가?

```
맴돌이전류 측정의 3가지 교란 요소:

[1] Lift-off 효과 (이격 거리):
  코일-피부 거리가 0.1mm만 변해도 신호가 크게 변함
  → VL53L0X ToF 센서로 실시간 거리 측정 후 보정

[2] 온도 드리프트:
  피부 전도도는 온도에 민감 (~2%/°C)
  체온 36.5°C에서 측정 vs 손 차가운 상태에서 측정 → 값 다름
  → MLX90614 적외선 온도계로 실시간 보정

[3] 개인차:
  피부 두께, 타입(건성/지성), 측정 부위(이마/팔뚝)마다 다름
  → 머신러닝으로 개인 캘리브레이션 모델 학습
```

### 7-2. 보정 수식

```
[Lift-off 보정]
  Z_보정 = Z_측정 / f(d)

  f(d): 거리-신호 감쇠 함수
  실험으로 구함: d = 1mm, 2mm, 3mm, 5mm에서 신호 측정 → 곡선 피팅

[온도 보정]
  Z_온도보정 = Z_측정 / (1 + β × (T - T_ref))

  β = 온도 계수 (약 0.015~0.025 /°C)
  T_ref = 기준 온도 (20°C 또는 측정 시작 온도)
  T = 현재 측정 온도
```

## 8. NDT와 피부 측정의 유사성 — 기술 혁신 포인트

### 8-1. NDT(비파괴검사)란?

```
Non-Destructive Testing: 파괴하지 않고 내부 결함 탐지

맴돌이전류 NDT 응용:
  - 항공기 동체 균열 검출
  - 금속 코팅 두께 측정
  - 파이프 부식 검사

대표 장비:
  - Elec Fine DMC-211: 4주파수 (31kHz~6.9MHz) 코팅 두께 측정
  - 가격: 수백만원
  - 대상: 금속, 합금
```

### 8-2. NDT → 피부 적용의 핵심 전환 발상

```
NDT 분야 관점:               SkinSense 관점:
─────────────────────────────────────────────────────
측정 대상: 금속 코팅         → 피부 각질층 (유사 구조!)
'노이즈' 로 간주:            → '신호' 로 활용!
  전도도 변화에 민감한 현상    → 수분=전도도 변화가 측정 목표
코팅 (비전도) + 금속 (전도)  → 각질층 (저전도) + 진피 (고전도)
다주파수로 두께 측정          → 다주파수로 깊이별 수분 측정
```

**이것이 특허의 핵심 창의성!**
> NDT에서 '오차'로 버려지던 전도도 민감도를,
> 피부 수분 측정에서 '신호'로 활용하는 발상의 전환

### 8-3. 물리 구조 비교표

| 항목 | NDT 코팅 측정 | SkinSense 피부 측정 |
|------|-------------|-------------------|
| 상층 (저전도) | 페인트 코팅 (두께 10~500μm) | 각질층 (두께 10~20μm) |
| 하층 (고전도) | 금속 기판 (σ >> 10⁶ S/m) | 진피층 (σ ≈ 0.5~2 S/m) |
| 측정 목적 | 코팅 두께 | 각질층 수분/전도도 변화 |
| 노이즈 요소 | 전도도 변화 | 이격 거리, 온도 |
| 주파수 범위 | 31kHz ~ 6.9MHz | 1MHz ~ 100MHz |
| 핵심 차이 | 전도도 변화 = 오차 | 전도도 변화 = **신호** |

## 9. 핵심 수식 정리

### 수식 1: 스킨 효과 침투 깊이

$$\delta = \frac{1}{\sqrt{\pi f \mu \sigma}}$$

- 주파수(f) 올리면 → δ 감소 → 더 얕은 층 측정
- 수분(σ) 증가하면 → δ 감소 → 실제로는 동일 주파수에서 더 얕게 침투

---

### 수식 2: 코일 공진 주파수

$$f_0 = \frac{1}{2\pi\sqrt{LC}}$$

- 피부 수분 ↑ → 등가 C ↑ → f₀ ↓ (공진 주파수 감소)
- Phase 1 Arduino 방식: f₀ 변화를 주파수 카운터로 측정

---

### 수식 3: 임피던스

$$Z = R + jX_L = R + j(2\pi f L)$$
$$|Z| = \sqrt{R^2 + (2\pi fL)^2}$$

- 피부 접근 → L 감소 → |Z| 감소
- 수분 증가 → σ 증가 → 맴돌이전류 강해짐 → 코일 L 더 감소

---

### 수식 4: 각질층 순수 신호 추출 (특허 청구항 2)

$$S_{sc} = Z_1 - (\alpha \times Z_2)$$

- Z₁: 고주파(100MHz) 측정값 (각질층 주도)
- Z₂: 저주파(1MHz) 측정값 (진피층 주도)
- α: 가중치 (0.1~0.3, 실험으로 최적화)

---

### 수식 5: 온도 보정

$$Z_{corrected} = \frac{Z_{measured}}{1 + \beta(T - T_{ref})}$$

- β = 0.015~0.025 /°C (온도 계수)
- T_ref = 기준 온도 (20°C 또는 실험 시작 온도)

---

### 수식 6: Lift-off 보정 (거리 보정)

$$Z_{corrected} = \frac{Z_{measured}}{f(d)}$$

실험으로 구한 f(d) 예시:
$$f(d) = e^{-k \cdot d}$$   또는  $$f(d) = \frac{1}{(1 + k \cdot d)^n}$$

- d: 코일-피부 거리 (mm)
- k, n: 실험 피팅 파라미터
