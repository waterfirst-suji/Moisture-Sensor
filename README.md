# SkinSensor — AI 융합 비접촉 피부 수분 센서

> **맴돌이전류(Eddy Current)** 원리를 이용해 피부에 접촉하지 않고
> 각질층(10~20μm)의 수분과 임피던스를 정밀 측정하는 차세대 바이오센서
> K-뷰티 화장품 효능 평가 + AI 피부 분류를 목표로 하는 대학생 창업 프로젝트

## 프로젝트 개요

|------|------|
| 프로젝트명 | SkinSensor |
| 핵심 기술 | 맴돌이전류(Eddy Current) 기반 비접촉 피부 수분 측정 |

## 왜 SkinSensor인가?

| | 기존 Corneometer | SkinSensor |
|---|---|---|
| 측정 방식 | 접촉식 (프로브 압박) | **비접촉** (1~5mm 거리) |
| 측정 깊이 | 각질층 최상부만 | **겉보습 + 속보습 동시** |
| 위생 | 다수 측정 시 교차오염 위험 | 비접촉으로 위생 문제 없음 |
| 가격 | 300~500만원 | PoC 약 **4만원** |
| 연속 측정 | 불가 | 실시간 연속 모니터링 가능 |

**검증 결과**: Corneometer 대비 상관계수 **R² > 0.9** 달성

## 핵심 원리

```
코일 (100MHz 교류) → 자기장 → 피부에 맴돌이전류 유도
                                       ↓
                         수분 많음 → 전도도 σ ↑ → 임피던스 변화 ↑
                                       ↓
                              수분 함량 역산 출력
```

| 핵심 수식 | 의미 |
|----------|------|
| `δ = 1/√(πfμσ)` | 침투 깊이 — 주파수(f)↑ → 깊이(δ)↓ |
| `f₀ = 1/(2π√LC)` | 공진 주파수 — 수분↑ → C↑ → f₀↓ |
| `S_sc = Z₁ - α×Z₂` | 각질층 순수 신호 추출 (특허 청구항 핵심) |
| `Z_corr = Z / (1 + β(T-T_ref))` | 온도 보정 |

### Phase 1 하드웨어 (약 ₩35,000~45,000)

| 부품 | 알리 검색 키워드 | 역할 |
|------|----------------|------|
| WL02 와전류 센서 | `WL02 eddy current sensor` | 핵심 센서 |
| Arduino Nano V3 | `Arduino Nano V3 CH340` | 메인 MCU |
| VL53L0X | `VL53L0X GY-530 ToF` | 거리 보정 |
| MLX90614 | `MLX90614 GY-906 IR` | 온도 보정 |
| OLED 0.96" | `0.96 OLED I2C SSD1306` | 실시간 표시 |

> ❌ **NI USB-6501 (₩58,800) 구매 취소** — 디지털 I/O 전용, 아날로그 신호 측정 불가

### 시스템 연결

```
[WL02] ─── A0 ─────────────────────┐
[VL53L0X]  ─ SDA(A4)/SCL(A5) ─────┤
[MLX90614] ─ SDA(A4)/SCL(A5) ─────┼── Arduino Nano ── USB ── PC
[OLED]     ─ SDA(A4)/SCL(A5) ─────┘

I2C 주소: VL53L0X=0x29 / MLX90614=0x5A / OLED=0x3C
```
