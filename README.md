# SkinSense — AI 융합 비접촉 피부 수분 센서

> **맴돌이전류(Eddy Current)** 원리를 이용해 피부에 접촉하지 않고
> 각질층(10~20μm)의 수분과 임피던스를 정밀 측정하는 차세대 바이오센서
> K-뷰티 화장품 효능 평가 + AI 피부 분류를 목표로 하는 대학생 창업 프로젝트

## 왜 SkinSense인가?

```
[기존 Corneometer의 한계]                [SkinSense의 혁신]
──────────────────────────────────────────────────────────
접촉식 → 프로브 압력으로 측정값 왜곡     비접촉 1~5mm 거리 측정
```

## 핵심 기술 원리 (3줄 요약)

1. **맴돌이전류**: 코일에 100MHz 교류 전류 → 변화하는 자기장 → 피부(도체) 내부에 소용돌이 전류 유도
2. **스킨 효과**: 높은 주파수일수록 침투 깊이(δ) 얕아짐 → `δ = 1/√(πfμσ)` → 100MHz에서 δ≈35μm → 각질층만 선택적 측정
3. **임피던스 변화**: 피부 수분 많음 → 전도도 σ 증가 → 맴돌이전류 강해짐 → 코일 임피던스 변화 → 수분 역산

## 빠른 시작 (Quick Start)

### Phase 1 하드웨어 준비물 (약 ₩35,000~45,000)

| 부품 | 알리 검색 키워드 | 용도 |
|------|----------------|------|
| **WL02 와전류 센서** | `WL02 eddy current sensor` | 핵심 센서 — 맴돌이전류 측정 |
| **Arduino Nano V3** | `Arduino Nano V3 CH340` | 메인 MCU |
| **VL53L0X** | `VL53L0X GY-530 ToF` | 코일-피부 거리 보정 |
| **MLX90614** | `MLX90614 GY-906 IR` | 피부 온도 보정 |
| **OLED 0.96"** | `0.96 OLED I2C SSD1306` | 실시간 값 표시 |
| **브레드보드 + 점퍼선** | `breadboard 830 jumper wire` | 납땜 없이 조립 |

> ❌ **NI USB-6501 (₩58,800) 구매 취소** — 디지털 I/O 전용이라 아날로그 신호 측정 불가
> ✅ 대신 Arduino Nano (~₩5,000) 사용 → 약 ₩54,000 절감 + 성능 우수

### 시스템 연결 구조

```
[WL02 와전류 센서] ─── A0 ─────┐
                               │
[VL53L0X 거리 센서] ─ SDA/SCL ─┤
[MLX90614 온도 센서] ─ SDA/SCL ─┼── Arduino Nano ─── USB ─── PC (Python)
[OLED 디스플레이] ─── SDA/SCL ─┘
                                         │
                               실시간 CSV + 그래프

I2C 주소: VL53L0X=0x29 / MLX90614=0x5A / OLED=0x3C (충돌 없음)
```

### 4주 실험 로드맵

| 주차 | 목표 | 성공 기준 |
|------|------|----------|
| Week 1 | 하드웨어 조립 + 알루미늄 호일 테스트 | 1cm 거리에서 ADC 값 ±50 이상 변화 |
| Week 2 | I2C 센서 추가 + 4채널 동시 수집 | 100ms 주기 안정적 CSV 출력 |
| Week 3 | 건조/물/식염수 전도도 비교 | 식염수 > 물 > 건조 순서 확인 |
| Week 4 | **보습 크림 PoC** — 도포 전/후 측정 | t-test p < 0.05 → 논문 데이터! |

## 핵심 수식

| 수식 | 의미 |
|------|------|
| `δ = 1/√(πfμσ)` | 스킨 효과 침투 깊이 — 주파수(f)↑ = 깊이(δ)↓ |
| `f₀ = 1/(2π√LC)` | LC 공진 주파수 — 수분↑ = C↑ = f₀↓ |
| `S_sc = Z₁ - α×Z₂` | 각질층 순수 신호 추출 (특허 청구항 1) |
| `Z_corr = Z / (1 + β(T-T_ref))` | 온도 보정 (β = 0.015~0.025/°C) |
